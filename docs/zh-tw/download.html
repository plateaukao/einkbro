下載 EinkBro

GitHub Releases

直接從 GitHub 下載最新的穩定版 APK。提供多種架構（arm64-v8a、armeabi-v7a、x86、x86_64）。

F-Droid

透過 F-Droid 自由軟體儲存庫安裝。更新時間可能較 GitHub releases 稍有延遲。

快照／Nightly 版

取得 CI 管線最新的開發版本。包含最新功能，但穩定度可能較低。

安裝方式

  1. 下載符合您裝置架構的 APK（大多數 E-Ink 裝置使用 arm64-v8a）
  2. 在裝置上前往「設定 → 安全性」並啟用「未知來源」（或允許從檔案管理員／瀏覽器安裝）
  3. 開啟下載的 APK 檔案並點擊「安裝」
  4. 安裝完成後，開啟 EinkBro 即可開始瀏覽

提示：若不確定您的裝置是哪種架構，先下載 arm64-v8a 版本即可 — 此版本適用於大多數現代 E-Ink 裝置，包括 Boox、Kobo（已安裝 Play 商店）以及 Kindle（透過側載安裝）。

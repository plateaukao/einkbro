使用指南

概觀

EinkBro 是專為 E-Ink 裝置設計的瀏覽器。隨著 E-Ink 螢幕越來越快、越來越強大，越來越多人在電子閱讀器上瀏覽網頁。但主流瀏覽器充斥著動畫、淡入淡出與各種 UI 特效，在 E-Ink 螢幕上的表現糟糕。

EinkBro 遵循兩大核心設計原則：

  • 減少重繪次數 — 更少的螢幕重新整理，帶來更順暢的使用體驗
  • 縮小重繪區域 — 更小的更新範圍，減少殘影

最終成果是一款具備高對比圖示、零動畫，並支援觸控區翻頁、音量鍵、閱讀模式、直書版型與 AI 內容互動的瀏覽器。

首次啟動

第一次開啟 EinkBro 時，您會看到下方的工具列，上面是可自訂的動作圖示。預設配置如下：

  • 手機：新分頁、觸控、閱讀模式、重新整理、上一頁、書籤、分頁計數、輸入網址、設定
  • 平板：標題、新分頁、觸控、閱讀模式、重新整理、上一頁、書籤、分頁計數、設定

點擊「設定」圖示（漢堡選單）即可開啟動作選單，存取所有瀏覽器功能。

提示：「設定」畫面頂端有搜尋列，輸入任何設定名稱即可直接跳到該項目。搜尋同時支援當地語系與英文關鍵字，無論應用程式使用何種語言皆可使用。

提示：變更應用程式語言可立即套用，無需重新啟動應用程式，且開啟的分頁會被保留。

工具列自訂

開啟「動作選單 → 工具列圖示」（或將「工具列設定」按鈕加入工具列）。勾選任一項目即可加入工具列。拖曳右側把手可重新排序。

工具列位置可於「設定 → 工具列」設為下方、上方、左側或右側。

動作選單

點擊工具列上的「設定」（漢堡）圖示即可開啟動作選單，裡頭以可捲動的列排列了所有瀏覽器功能。

第一列：快速動作

重點 — 檢視與管理文字重點 設為首頁 — 將目前網址設為首頁 開啟首頁 — 前往首頁 關閉分頁 — 關閉目前分頁（若為最後一個則關閉應用程式） 結束 — 離開 EinkBro

分享與儲存

可收合區塊，包含兩列分享與儲存動作：

接收連結 — 透過區域網路從另一裝置接收網址 加入書籤 — 將目前頁面加入書籤 加到主畫面 — 將捷徑加到 Android 主畫面 以其他應用程式開啟連結… — 以其他應用程式（YouTube、地圖等）開啟網址 複製連結 — 將網址複製到剪貼簿 分享連結 — 將網址分享到其他應用程式 傳送連結 — 傳送網址至另一台執行 EinkBro 或 Shakira 的裝置 Instapaper — 儲存至 Instapaper 稍後閱讀 — 本機的「稍後閱讀」 EPUB — 開啟或儲存為 EPUB 儲存為 PDF — 匯出為 PDF，紙張大小可設定 儲存為 MHT — 將目前頁面封存為單一 .mht 檔

內容調整

閱讀模式

閱讀模式會移除頁首、側欄、頁尾與廣告，只留下主要文章內容。非常適合在 E-Ink 螢幕上長時間閱讀。

  • 透過工具列的「閱讀模式」按鈕或「動作選單 → 閱讀模式」啟用
  • 長按工具列上的「字級大小」按鈕也會進入閱讀模式
  • 可在「設定 → 介面 → 閱讀模式留白」中自訂留白
  • 在「設定 → 其他」啟用「保留額外內容」可保留數學公式（MathJax／KaTeX）與樣式
  • 閱讀模式擁有獨立的工具列設定，包含：旋轉、全螢幕、粗體字、字級大小、觸控、目錄、頁面資訊、設定、關閉分頁

分割畫面

分割畫面會在目前頁面旁開啟第二個網頁面板。適合邊看新聞清單邊讀文章，或在閱讀內容的同時使用 AI 對話。

  • 透過「動作選單 → 分割畫面」或長按任一連結即可啟用
  • 分割面板擁有自己的迷你工具列（由上而下）：
    • 方向 — 切換直向／橫向分割。長按可交換兩個面板。
    • 雙畫面連結 — 啟用後，主面板的連結會在第二面板開啟
    • 同步捲動 — 同步兩面板的捲動（適合比對翻譯）
    • 字級 +/− — 調整分割面板的字級
    • 關閉 — 關閉分割畫面

翻譯

EinkBro 提供多種翻譯方式：

逐段翻譯

以內嵌方式逐段翻譯，將翻譯顯示於原文旁。可透過工具列的「逐段翻譯」動作啟用，或長按「翻譯」按鈕以選擇模式。提供者：Google Translate、DeepL、OpenAI、Gemini。

個別站台翻譯模式

翻譯設定對話框（長按翻譯）可為每個站台分別記住偏好的翻譯模式 — 當某個站台適合 DeepL 逐段對照、另一個站台適合 Gemini 原地翻譯時非常實用。下次造訪時會自動恢復先前選擇的模式。

原地翻譯

使用 OpenAI 或 Google Gemini 進行原地翻譯，直接替換原文內容。可長按「翻譯」按鈕並選擇原地模式。

整頁翻譯

透過 Google Translate 翻譯整個頁面。可透過「動作選單 → 翻譯」或工具列的「翻譯」按鈕啟用。

圖片翻譯

翻譯圖片中的文字。結果會快取到磁碟，加速後續存取。長按翻譯圖示可批次翻譯剩餘所有圖片。需於「設定 → 其他 → 翻譯圖片 API 金鑰」設定 API 金鑰。

雙字幕

於 YouTube 影片顯示兩種語言的字幕。可在「設定 → 其他 → YouTube 雙字幕」中設定。

EPUB 匯出

將網頁文章儲存為 EPUB 電子書檔案，包含圖片與目錄。

  • 透過「動作選單 → EPUB」啟用
  • 對話框包含「開啟現有 EPUB」與「儲存新 EPUB」分頁
  • 可將多個網頁儲存到同一份 EPUB，每頁成為一個章節
  • 可在目錄編輯器中拖曳重新排序章節
  • 儲存的 EPUB 包含 EinkBro 識別資訊，便於辨識
  • 為求最佳效果，建議內容能相容於閱讀模式

觸控翻頁

EinkBro 為 E-Ink 裝置打造的招牌功能。點擊螢幕左／右（或上／下）邊緣即可向前／向後翻頁，就像翻動電子書一樣。

  • 透過工具列「觸控翻頁」按鈕啟用
  • 電子書觸控模式 — 以原生觸控攔截實現直接點擊翻頁（不再使用 JS 注入）
  • 於「設定 → 手勢」設定觸控區域與動作
  • 可選擇左右或上下的觸控區配置
  • 長按音量鍵可暫時停用翻頁 5 秒
  • 音量鍵也可用於翻頁（於「快速切換」中啟用）
  • 在「設定 → 介面」中調整「換頁保留高度」以控制頁面重疊量

AI／ChatGPT 整合

直接在瀏覽器中透過 AI 與網頁內容互動。

與網頁對話

開啟分割畫面的 AI 聊天面板，詢問目前頁面的問題。可透過工具列「與網頁對話」按鈕或「動作選單 → 與網頁對話」啟用。

頁面 AI

對整頁內容執行 AI 動作（例如摘要、擷取重點）。使用於「設定 → GPT → 整頁內容 Prompt」中設定的 Prompt。

任務流程（Task Runner）

在目前頁面串聯多步驟 AI 工作流程 — 顯示為「頁面 AI」動作選單的第一項。可定義一連串 Prompt，針對頁面內容、選取文字或前一步的輸出進行處理。適合經常重複執行的「擷取 → 翻譯 → 摘要」等流程。

自由形式 Agent

整合於「與網頁對話」中，Agent 可處理開放式的後續對話，並持續存取頁面內容與先前輪次的紀錄。無需重複貼上上下文即可反覆分析，也可回呼頁面工具。

新聞主播 TTS

朗讀動作範本，以新聞主播風格的語音朗讀完整文章，使用 OpenAI 的 gpt-4o-mini-tts 模型與預先設定的語音指示。可於「設定 → GPT → 語音輸出指示」中調整。

自訂 GPT 動作

於「設定 → GPT → ChatGPT 動作定義」以自訂系統 Prompt 定義可重複使用的 AI 動作。這些動作會出現在文字選取的快顯選單，並可對所選文字或整頁內容觸發。

支援的提供者

  • OpenAI — 透過 API 金鑰使用 GPT-4.1 等模型
  • Google Gemini — 支援 Gemini 2.5 Flash 等
  • 自架／Ollama — 透過自訂主機網址，使用任何 OpenAI 相容伺服器

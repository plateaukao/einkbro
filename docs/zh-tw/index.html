EinkBro

專為 E-Ink 裝置打造的 Android 瀏覽器

使用指南 下載

功能特色

E-Ink 最佳化閱讀

點擊螢幕邊緣或使用音量鍵翻頁。支援閱讀模式、直書版型、自訂字型、粗體切換，並完全沒有動畫效果。

AI 整合

與網頁對話、以自由形式 Agent 執行多步驟「任務流程」、進行摘要，並以新聞主播語音朗讀文章。支援 OpenAI、Gemini、Ollama 與自訂伺服器。

翻譯

段落逐段對照翻譯、整頁翻譯，以及圖片文字翻譯。支援多種服務，包括 DeepL 與 Google。

儲存與匯出

匯出為包含章節與圖片的 EPUB、儲存為 PDF、封存為 MHT、擷取整頁截圖，或傳送至 Instapaper。

可自訂介面

40 個以上可設定的工具列動作，並支援拖曳排序。工具列可放在上方、下方或側邊。支援分割畫面、浮動按鈕與手勢導航。

隱私與瀏覽

內建廣告封鎖、JavaScript／Cookie 白名單、追蹤參數清除、無痕模式，以及網站個別的桌面模式。
